# CaveCraft
2D Minecraft clone for Android and Desktop. <br>
Written in Java using libGDX framework. <br>
## Build instructions
### Android
`./gradlew android:assemble` <br>
### Desktop
`./gradlew desktop:assemble` <br>
***
On Windows, use `gradlew.bat` instead of `./gradlew`
